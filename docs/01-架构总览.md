## 目录结构与职责

```
index.html          HTML 骨架，所有 DOM 结构定义于此
css/                纯样式，不含逻辑
js/
  config.js         全局变量占位（VIDEO_BASE, CHAPTERS），由 app.js 在运行时填充
  utils.js          无副作用纯函数工具（格式化、时长预加载）
  state.js          单例 VchapState，承载所有运行时状态
  player.js         VchapPlayer IIFE，封装 Video.js，对外暴露语义化 API
  controls.js       VchapControls IIFE，所有 UI 交互、事件绑定
  sidebar.js        VchapSidebar IIFE，章节列表的渲染与高亮
  app.js            顶层全局函数（vchapBootstrap、vchapSetMode 等），DOMContentLoaded 入口
vendor/             Video.js 本地副本，不修改
```

---

## 脚本加载顺序（关键）

```
vendor/video.min.js   ← Video.js 库，必须最先
config.js             ← 全局变量声明
utils.js              ← 纯工具函数
state.js              ← VchapState 对象
player.js             ← VchapPlayer（依赖 VchapState、VchapControls、VchapSidebar）
controls.js           ← VchapControls（依赖 VchapState、VchapPlayer）
sidebar.js            ← VchapSidebar（依赖 VchapState、VchapPlayer）
app.js                ← 入口，依赖以上全部
```

所有模块通过全局变量相互引用，**无模块化打包**。顺序错乱会导致引用未定义。

## 数据流

```
URL ?config=<json_url>
        │
        ▼ fetch()
    JSON 数据
        │
        ├─ VIDEO_BASE = data.videoBase
        └─ CHAPTERS   = data.chapters
                │
                ▼ vchapBootstrap()
        ┌───────────────────┐
        │  VchapState.init() │  初始化 durations / startTimes 数组
        └───────────────────┘
                │
        ┌───────┴────────────────────────────────┐
        │               │                        │
   VchapPlayer.init()  VchapControls.init()   VchapSidebar.render()
        │                │
        └────────────────┘
               │ 事件回调（play/pause/timeupdate/ended …）
               ▼
          VchapState 更新
               │
      ┌────────┴────────┐
      ▼                 ▼
VchapControls.        VchapSidebar.
updateProgress()     updateActive()
```

## 模式切换

```
vchapSetMode('full')    → VchapState.mode = 'full'
                          CSS class: page.mode-full
                          侧边栏隐藏，章节标记显示

vchapSetMode('chapter') → VchapState.mode = 'chapter'
                          CSS class: page.mode-chapter
                          侧边栏显示，章节标记隐藏
```

进度条 seek 逻辑根据 `VchapState.mode` 分支：
- `full` → `VchapPlayer.seekGlobal(t)` — 映射全局时间 → 章节 + 本地偏移
- `chapter` → `VchapPlayer.seekLocal(t)` — 直接 `vjs.currentTime(t)`

## 命名规范

| 类型 | 规范 | 示例 |
|------|------|------|
| 全局函数 | `vchapXxxYyy` camelCase | `vchapBootstrap`, `vchapSetMode` |
| 全局对象/模块 | `VchapXxx` PascalCase | `VchapState`, `VchapPlayer` |
| DOM 元素 id（新增） | `vchap-xxx-yyy` kebab-case | `vchap-res-wrap`, `vchap-res-btn` |
| CSS 类名（新增） | `vchap-xxx-yyy` kebab-case | `.vchap-res-opt`, `.vchap-chap-marker` |
