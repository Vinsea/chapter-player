# 分辨率切换实现说明

> 开发者文档 · story-deck-2
> 作者：vinsea · 2026-06-12

---

## 核心约定

**`qualities` 数组必须从低到高排列**：第一项为默认加载的低清版，最后一项为自动升级的目标高清版。代码不解析 label 文字，仅依赖数组位置。

```json
"qualities": [
  { "label": "720P",  "src": "ch01_720p.mp4"  },  // index 0 → 默认加载
  { "label": "1080P", "src": "ch01_1080p.mp4" }   // index -1 → 升级目标
]
```

---

## 设计目标

- JSON 配置每章节可选 `qualities` 数组，每项包含 `label` 和 `src`
- 控制栏右侧出现"画质"按钮，hover 弹出菜单，当前档高亮
- 切换时保留当前播放时间位置和播放/暂停状态
- 单分辨率（只有 `src`）或所有章节只有一档时，按钮自动隐藏

---

## 数据层（VchapState）

```
VchapState.currentQuality: string
  — 当前选中的 label（如 '1080P'）
  — 初始值由 _bindQuality() 设为第一个章节第一档的 label
  — '' 表示未初始化或单文件模式
```

---

## 播放层（VchapPlayer）

### `_srcForChapter(idx)` — 私有

```
伪代码：
  ch = CHAPTERS[idx]
  if ch.qualities 存在且非空:
    遍历 ch.qualities 找 label === VchapState.currentQuality
    找到 → 返回 VIDEO_BASE + 匹配项.src
    未找到 → 返回 VIDEO_BASE + ch.qualities[0].src  // 回退到第一档
  else:
    返回 VIDEO_BASE + ch.src
```

所有 `loadChapter` / `_doLoad` 内部调用 `_srcForChapter`，不再硬编码 `ch.src`。

### `setQuality(label)` — 公开

```
伪代码：
  if label === VchapState.currentQuality: return  // 无变化，忽略

  VchapState.currentQuality = label
  savedTime  = vjs.currentTime()
  wasPlaying = VchapState.isPlaying

  vjs.src({ src: _srcForChapter(currentIdx), type: 'video/mp4' })
  vjs.load()
  vjs.one('loadedmetadata', function():
    vjs.currentTime(savedTime)
    if VchapState.currentRate !== 1: vjs.playbackRate(VchapState.currentRate)
    if wasPlaying: vjs.play()

  VchapControls.syncQualityUI()
```

---

## UI 层（VchapControls）

### `_bindQuality()` — 私有，在 `init()` 中调用

```
伪代码：
  wrap = #vchap-res-wrap
  hasQualities = CHAPTERS 中至少一章有 qualities.length > 1
  if !hasQualities: wrap.style.display = 'none'; return

  firstWithQ = CHAPTERS 中第一个有 qualities 的章节
  VchapState.currentQuality = firstWithQ.qualities[0].label

  for each quality in firstWithQ.qualities:
    创建 .vchap-res-opt div
    click → VchapPlayer.setQuality(quality.label)
    追加到 #vchap-res-menu-inner

  更新 #vchap-res-btn 文字 = VchapState.currentQuality
```

### `syncQualityUI()` — 公开

```
伪代码：
  #vchap-res-btn.textContent = VchapState.currentQuality
  for each .vchap-res-opt:
    toggle class 'active' 当 data-label === VchapState.currentQuality
```

## CSS 行为

- `.vchap-res-wrap:hover .vchap-res-menu` — hover 显示菜单（同倍速菜单逻辑）
- `.vchap-res-opt.active` — 当前档高亮为 cyan 色

---

## 边界情况处理

| 情况 | 处理方式 |
|------|---------|
| 某章节无该分辨率 label | 回退到该章 `qualities[0]` |
| 全部章节只有 `src` | 按钮 `display:none` |
| 切换分辨率时章节正在加载 | `loadedmetadata` 回调保证 seek 在元数据就绪后执行 |
| 切换分辨率后切章 | `_doLoad` 始终调用 `_srcForChapter`，自动使用新 label |
