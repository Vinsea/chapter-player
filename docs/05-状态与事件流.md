## VchapState 变更时机

```
属性               何时变更
────────────────────────────────────────────────────────────
mode               vchapSetMode() 调用时
currentIdx         _doLoad() 开始加载新章节时
isPlaying          vjs 'play' / 'pause' 事件回调
isSeeking          进度条 mousedown=true / mouseup=false
durations[]        vchapLoadAllDurations() 串行预加载完成后
startTimes[]       同上
totalDuration      同上
durLoaded          所有章节时长加载完毕后设为 true
watchedSet         vjs 'ended' 事件触发后（markWatched）
transitionOn       过渡开关 toggle 点击时
currentRate        VchapPlayer.setRate() 调用时
currentQuality     VchapPlayer.setQuality() 调用时
```

## Video.js 事件 → UI 响应

```
vjs 'play'         → VchapState.isPlaying=true
                   → VchapControls.updatePlayIcon()
                   → VchapSidebar.updateActive()

vjs 'pause'        → VchapState.isPlaying=false
                   → VchapControls.updatePlayIcon()
                   → VchapSidebar.updateActive()

vjs 'ended'        → VchapState.markWatched(currentIdx)
                   → VchapSidebar.refresh()
                   → VchapPlayer.loadChapter(next, true)  // 自动下一章

vjs 'timeupdate'   → VchapControls.updateProgress()       // 进度条刷新（约 4 次/秒）

vjs 'loadedmetadata' → VchapState.durations[currentIdx] = vjs.duration()（章节模式时补充时长）

vjs 'waiting'      → VchapControls.setLoading(true)
vjs 'playing'      → VchapControls.setLoading(false)
vjs 'canplay'      → VchapControls.setLoading(false)

vjs 'volumechange' → VchapControls.syncVolumeUI()
vjs 'ratechange'   → VchapControls.syncRateUI()
vjs 'error'        → VchapControls.showError(msg)
```

## 用户交互 → 系统响应

```
点击进度条
  mousedown → VchapState.isSeeking=true
  mousemove → _doSeek(pct)
              full+durLoaded → VchapPlayer.seekGlobal(t)   可能触发章节切换
              chapter       → VchapPlayer.seekLocal(t)
  mouseup   → VchapState.isSeeking=false, 最终 _doSeek

点击章节（侧边栏 / 进度条标记）
  full+durLoaded → VchapPlayer.seekGlobal(startTimes[idx])
  chapter       → VchapPlayer.loadChapter(idx, true)

键盘 Space/K   → VchapPlayer.togglePlay()
键盘 ←→       → VchapPlayer.seekGlobal / seekLocal ±5s
键盘 ↑↓       → VchapPlayer.setVolume ±0.1
键盘 F        → VchapPlayer.requestFullscreen / exitFullscreen
键盘 M        → VchapPlayer.setMuted(!getMuted)

倍速菜单选项   → VchapPlayer.setRate(s) → vjs 'ratechange' → VchapControls.syncRateUI()
画质菜单选项   → VchapPlayer.setQuality(label) → reload + restore pos → VchapControls.syncQualityUI()
过渡开关       → VchapState.transitionOn toggle
```

## 启动流程（伪代码）

```
DOMContentLoaded
  params = URL 解析 ?config=
  fetch(configUrl)
    → VIDEO_BASE = data.videoBase
    → CHAPTERS   = data.chapters
    → vchapBootstrap()
        VchapState.init()
        VchapPlayer.init()          // 创建 vjs，绑定 vjs 事件
        VchapControls.init()        // 绑定所有 DOM 事件
        VchapSidebar.render()       // 渲染章节列表
        VchapPlayer.loadChapter(0)  // 加载第一章
        vchapLoadAllDurations()     // 串行预加载所有时长
          当全部完成:
            VchapState.durLoaded = true
            VchapControls.renderMarkers()
            VchapSidebar.refresh()
            更新 nav-badge 显示总时长
    → vchapHideLoading()
```
